# FLogger 1.0.1 发布
2017-02-06
### v1.0.1 版本包含以下改进和 bug 修复：
#### 改进：
* 新增输出到控制台的功能，方便开发调试。详情参看配置：  
```
# 是否输出到控制台(默认为false)
CONSOLE_PRINT = false
```
  
#### bug修复：
* 无 
  
可下载文件已经发布到 [Maven Central Repository](http://repo1.maven.org/maven2/com/github/cyfonly/flogger/1.0.1/)。
